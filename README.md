TABI3A is a backend REST API designed to promote eco-tourism and natural heritage in Tunisia.

## Features
- JWT authentication with role-based access control
- Management of national parks, species, and locations
- Weather-based smart trip recommendations
- RESTful API documented with Swagger
- Dockerized for easy deployment

## Tech Stack
- FastAPI
- SQLAlchemy
- SQLite
- Docker
- Swagger (OpenAPI)
